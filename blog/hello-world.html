How This Site Works

This entire site is built with plain HTML, CSS, and JavaScript — no frameworks, no build tools, no dependencies. It's hosted for free on GitHub Pages.

Some fun technical details:

  • The shared navigation and footer are loaded as HTML fragments via fetch() and injected with JavaScript.
  • Dark/light mode is persisted in localStorage and respects your system preference by default.
  • Scroll animations use the IntersectionObserver API — no libraries needed.
  • The pixel-art visitor map on the home page uses a <canvas> element and a free IP geolocation API.
The best way to learn is to build, and the best way to share is to write.
